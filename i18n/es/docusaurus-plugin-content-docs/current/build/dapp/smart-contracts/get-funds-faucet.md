# Obtener Fondos de Prueba del Faucet de la Red de Pruebas Fuji

El [Faucet de la Red de Pruebas Fuji](https://faucet.lux.network) envía LUX a direcciones de la cadena X o de la cadena C
para ayudarte a hacer pruebas. (Este LUX de red de pruebas no tiene valor.)

Podemos obtener un "goteo" de LUX del faucet Fuji. Pega la dirección en el
sitio web del [faucet Fuji](https://faucet.lux.network). Estos LUX son para la red de pruebas Fuji
y no tienen valor monetario.

![Solicitando LUX](/img/fuji-workflow/faucet1.png)

El faucet enviará algunos LUX a la dirección y devolverá un ID de transacción
(txID). Este txID se puede usar con el Explorador de la Red de Pruebas Fuji para obtener más información sobre
la transacción.

![Recibiendo LUX](/img/fuji-workflow/faucet2.png)
